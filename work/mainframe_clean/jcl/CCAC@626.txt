//*=====================================================================
//*    CCAC@626   1099 FILE PREPARATION
//*
//* SYSTEM NARRATIVE
//* -----------------
//* THIS JOB CREATES THE 1099 MASTER UPDATES FILE WHICH CONTAINS ONLY
//* THE DELETE AND CHANGE TRANSACTIONS.  IT ALSO MAINTAINS THE CORPORATE
//* FILE BY COPYING AND REFORMATTING FROM THE ACCOUNTING SUPERVISOR
//* APPROVED 1099 USER MASTER FILE ALL RECORDS CONTINAING "B" IN COLUMN
//* 205.  CCAC@626 COPIES THE REJECT AND MANUAL ISSUES FILE TO A GDG,
//* EXTRACTS THE MASTER RECORDS THAT HAVE BEEN FLAGGED AS CHANGE OR
//* DELETE AND CREATES HEADER AND TRAILER RECORDS FOR THE MASTER UPDATES
//* FILE.  THE MANUAL ISSUES FILE (USED IN CCAC@625) IS THEN EMPTIED.
//*
//* PROCESSING SCHEDULE
//* ---------------------
//* THIS JOB IS RUN UNDER CA-7. IT RUNS MONTHLY ON THE FOURTH WORKDAY.
//* IT IS DEPENDENT ON JOB CCAC626X SUBMITTED BY ACCOUNTING AND
//* SUCCESSFUL COMPLETION OF JOB CCAC@625. IT HAS ONLY ONE PROCESSING
//* SCHEDULE.
//*
//* SPECIAL INSTRUCTIONS
//* ---------------------
//* VERIFY ACCOUNTING SUBMITTED JOB CCAC625X TO SATISFY DEPENDENCY.
//*
//* RERUN/RESTART INSTRUCTIONS
//* --------------------------
//* TO RERUN THIS JOB THE ACCOUNTING ANALYST WILL SUBMIT A REQUEST.
//* TO RESTART THIS JOB, EXECUTE FROM THE BEGINNING.
